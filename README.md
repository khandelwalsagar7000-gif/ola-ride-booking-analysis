# Ola Ride Booking Analysis Dashboard

## Project Overview
This project analyzes ride booking data using Power BI and SQL to identify booking trends, cancellations, revenue patterns, and customer rating behavior.

## Tools Used
- Power BI
- SQL
- Excel

## Dataset Details
- Total Bookings: 103,024
- Booking Value: 35M

## Key Analysis Performed
- Booking status analysis
- Vehicle type analysis
- Revenue analysis
- Cancellation analysis
- Customer ratings analysis

## Key Insights
- Identified booking and cancellation trends
- Analyzed revenue contribution across ride categories
- Evaluated customer rating patterns
- Supported decision-making through interactive dashboard reporting
